# Hello Word
> Pero sin ser un solo archivo

## Descripción
En el directorio `important_helloWorld` se encuentran muchos otros subdirectorios relacionados con
una misma función: mostrar un _hello world_.
### Primer Ejemplo
Para ver el primer ejemplo, no es mas que ejecutar el archivo _simple_helloWorld.py_  
Desde una terminal deberia verse de esta forma:

> \important_helloWorld> python simple_helloWorld.py

### Segundo Ejemplo
Para ver el segundo ejemplo, basta con ejecutar desde la terminal:
> \important_helloWorld> python manage.py runserver

Y visitar la IP local de su maquina, bien sea `127.0.0.1` con el puerto `8000`  en la dirección de `/hello_world`.  
Algo como: `127.0.0.1:8000/hello_world`
